PROJECTS
DoorKnock

DoorKnock is a Windows 10 APP (UWP) designed to be paired with a companion app on a smartphone. The desktop application handles incoming "knocks" to notify the user of someone requiring their attention within their local network.

I had recently obtained a pair of closed-back headphones, limiting my ability to hear my surroundings. Due to this I started working on an application with communication within a household in mind. I completed a rough prototype of how the desktop application would work, however due to lack of hardware, the project was indefinitely postponed.

Nevertheless, this gave me some wonderful insights into the development process of a Windows 10 app and the Universal Windows Platform. The app was mainly coded in C# with ASP.NET CORE as the main framework.

PalicoDB

PalicoDB was created as a sandbox project with the intent of learning more about the Angular framework. The idea was to create a database website for the popular game Monster Hunter World. It would keep track of information relating to items ingame, as well as adding some extra functionality if time and circumstance permitted it so.

The site itself has an Angular front-end, serviced by a Node.js (Express) back-end. SQL Server is used to store all information (locally at the moment). In it's current iteration, the project supports CRUD operations on Items facilitated by a REST API.

This sandbox project gave me a lot experience with Angular, letting me learn a bit on how front-end frameworks operate and what advantages they bring to web development.
